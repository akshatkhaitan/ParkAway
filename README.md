# ParkAway
## ParkAway - OOPS Project
This project was made as a part of the Object Oriented Programming Project in my Sophomore Year. 
## Implmented Using
Frontend : React JS <br />
BackEnd : SpringBoot <br />
DataBase : Mongo DB Atlas
